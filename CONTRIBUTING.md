# Contributing

Thanks for your interest in contributing to `parser-go`.

## Before you start

- Check for an existing issue or pull request first.
- For security issues, follow the process in [`SECURITY.md`](./SECURITY.md).

## Development

```sh
go test ./...
go vet ./...
go build ./...
```

## Pull requests

Please keep pull requests focused and include:

- A short summary of the change
- Tests covering new or changed behavior
- Updated docs if applicable
