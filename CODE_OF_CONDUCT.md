# Code of conduct

We want issues, pull requests, and benchmark discussions in `github.com/sagaragas/parser-go` to stay useful, respectful, and technical.

## Expectations

When you participate in the project, please:

- be respectful, direct, and specific
- focus on the code, the data, or the decision in front of you instead of the person behind it
- avoid harassment, discrimination, intimidation, or dogpiling
- avoid posting private data such as secrets, exploit details, raw homelab logs, or personal information in public threads
- accept maintainer moderation when a thread has stopped being productive

## Scope

This applies to project issues, pull requests, discussions, wiki edits, and any other public space tied to this repository.

## Enforcement

Maintainers may edit, hide, lock, or remove content that breaks these expectations, and they may restrict participation when needed. If you need to report a conduct concern, contact the maintainer through the profile attached to the main GitHub repository and keep the report as private as possible.
